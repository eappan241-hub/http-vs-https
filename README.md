# Difference Between HTTP and HTTPS


## HTTP
- Stands for HyperText Transfer Protocol
- Data is sent in plain text
- Less secure and vulnerable to attacks
- Uses Port 80 for communication
- Does not use SSL/TLS certificates
- URL starts with `http://`

## HTTPS
- Stands for HyperText Transfer Proocol Secure
- Data is sent in encrypted form
- More secure and protects user data
- Uses Port 443 for communication
- Uses SSL/TLS certificates for security
- URL starts with `https://` and shows a padlock icon
